# task-tracker
Simple command-line task tracker in Python

## Features
- Add, view, and manage tasks from the command line
- Simple and intuitive interface
- Easy to use and extend

## Installation
Clone the repository and run the task tracker:
```bash
git clone https://github.com/9448920875/task-tracker.git
cd task-tracker
python task_tracker.py
```

## Usage
Get started by running the application and following the on-screen instructions.
